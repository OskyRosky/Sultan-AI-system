# 04 Roadmap

## Etapas

| Etapa | Nombre | Resultado esperado |
| --- | --- | --- |
| 0 | Inicio | Documentación y estructura base. |
| 1 | Setup Mac + Repo | Entorno local y estándares mínimos. |
| 2 | Investment Memo Final | Tesis, límites y criterios de éxito. |
| 3 | Data Platform MVP | Datos BTC/ETH en raw y curated. |
| 4 | Data Quality + Lineage | Validaciones y trazabilidad de datos. |
| 5 | Feature Layer MVP | Features iniciales reproducibles. |
| 6 | Research Notebook Base | Investigación controlada. |
| 7 | Primeras Estrategias | Baselines simples y explicables. |
| 8 | Backtesting Robusto | Simulación con costos, slippage y métricas. |
| 9 | Regime Detection | Clasificación inicial de regímenes. |
| 10 | Signal Fusion | Combinación controlada de señales. |
| 11 | Risk Engine | Veto, límites y kill-switch. |
| 12 | Paper Trading | Simulación operativa en tiempo real. |
| 13 | Dashboard + Monitoreo | Visibilidad completa del sistema. |
| 14 | Live-Small + Escalamiento | Exposición mínima bajo controles estrictos. |

## Criterio de avance

Ninguna etapa debe saltarse si introduce riesgo operativo, falta de trazabilidad o ejecución no autorizada.
